GIT COMMANDS

	1. git init

	creates for you hidden folder that holds all the files to be sent to git

	2. git add .

	adds all the files in the .git folder. Excludes all ignored files

	3. git add *

	adds all the files in the .git folder. Including all ignored files

	4. git add <filename>

	adds that specific file name in your .git folder

	5. git commit -m 'description'
	
	describes what is being committed (file being committed)

	<.................. for first time users........................>
		 to set up your account use the following

		git config --global user.email "user@example.com"
		
            	git config --global user.name "user name"

	6. git remote add origin.

	7.git push -u origin master
